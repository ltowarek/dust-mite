# Variants

dust-mite is developed as multiple hardware + software variants.

## Available variants

- [Copper](variants/copper.md) — ESP32-S3 based setup on a basic 4WD skid-steering chassis.
- [Iron](variants/iron.md) — Raspberry Pi 2 + ESP32-S3 split architecture with RPi2-hosted control/streaming stack and serial hardware control bridge.
